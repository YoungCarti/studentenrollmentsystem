# Student Enrollment Management System (SEMS)

A Java EE web application for managing student enrollments.

## Project Status
- **Phase**: Frontend Implementation (Week 1)
- **Frontend**: JSP + Vanilla CSS (Modern Dark Theme)
- **Backend**: Pending (Structure ready)

## How to Run (NetBeans)
1. Open this folder in NetBeans.
2. Right-click project > **Clean and Build**.
3. Right-click project > **Run**.
4. Application opens at `http://localhost:8080/studentenrollment/`.

## Manual Navigation (Preview Mode)
Since the backend authentication logic is not yet implemented, use these URLs to preview the protected pages:
- **Login**: `/auth/login.jsp`
- **Student Dashboard**: `/student/dashboard.jsp`
- **Admin Dashboard**: `/admin/dashboard.jsp`
